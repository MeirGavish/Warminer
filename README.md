### Warminer ###

A mobile clicker tower defense game developed in Unity with server side data in Firebase.
Developed as a project for a game development course at Hands On Games.

The concept is that the clicker and tower defense mechanics feeding into each other. 
You gain clicker "metal" currency by mining it and you can use it to buy upgrades for the tower defense. 
From the tower defense you gain currency by defeating enemies and you can use it to buy upgrades for the clicker.
